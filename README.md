## 📌 Project Overview

This project showcases an interactive Sales & Profit Performance Dashboard built in Microsoft Power BI. The dashboard provides insights into sales, profit, product performance, customer segments, and regional trends through interactive visualizations, custom tooltips, bookmarks, and slicers. It is designed to help business users monitor performance and make data-driven decisions.

---

## 🎯 Project Objectives

- Analyze overall sales and profitability.
- Monitor product and regional performance.
- Identify top-performing products.
- Compare sales across customer segments.
- Enable interactive report navigation using bookmarks.
- Provide detailed insights using custom tooltips.

---

## 📈 Business Insights

- Consumer segment contributes the highest share of sales.
- Furniture generates the highest sales among categories.
- Central region records the highest profit.
- A small number of products contribute significantly to total revenue.
- Interactive filters help analyze performance by region, state, and customer segment.

## 🚀 How to Use

1. Download the `.pbix` file.
2. Open it using Microsoft Power BI Desktop.
3. Refresh the dataset if required.
4. Use the slicers and bookmarks to explore the dashboard.
5. Hover over visuals to view custom tooltips.
